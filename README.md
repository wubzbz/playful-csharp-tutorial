## 这是什么？

一套**从零开始**的 C# 编程教程。用人话讲清楚每一个概念，拒绝黑话与机翻。

我们在讲 C#，但我们不是只聊 C# —— 音乐、魔法、烹饪、水浒传……话题随时可能跑偏，但知识一定到位。
