# A Playful C# Tutorial

这是一个既简单又有趣的C#语言教程。许多教程遣词造句十分专业、描述抽象，或者默认读者具有相当的相关知识储备（比如计算机组成、操作系统等），对于没有相关基础或者不关心编程以外内容的小白不太友好。所以我正在做一个力求通俗易懂、即学即用的C#教程！

!!! warning

    本教程尚未完成，将持续更新。
