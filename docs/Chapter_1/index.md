# 第1章：认识C#中的类型（初级篇）


欢迎来到由各种各样的数据组成的世界！

当我们编写、调试和运行程序时，我们实际上处理了从数字到图像的各类数据。在游戏中，玩家按下WASD键控制人物的移动，人物的坐标（小数）是数据。使用社交媒体时，给喜欢的内容点赞，点赞数（整数）也是数据。登录账号时，需要输入密码，密码（字符串）也是数据。

本章我们将了解C#中常用的内置类型。

!!! info "C#世界观"

    你会逐渐意识到，C#是一个构建在 **类型** 之上的世界。从现在开始，我们先逐一了解C#为我们提供的内置类型；然后，我们再学习如何基于这些简单的类型设计并构建更加复杂的类型。

!!! info "内置类型和用户定义类型"

    某日，你想下厨做一个蛋糕。在准备原材料的时候，你会考虑到商店买一些面粉、鸡蛋、食用油和盐；但你绝对不会打算买一些小麦种子回来种，或者买一些鸡苗回来养！<br>
    内置类型就是C#为我们提供好的原材料。我们可以方便地使用这些简单的内置类型制作出结构更复杂、功能更丰富的类型。事实上，这就是第1~3章的主题！
